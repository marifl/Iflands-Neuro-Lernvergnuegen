# Internes Vorwärtsmodell

Ein internes Vorwärtsmodell schätzt die erwarteten sensorischen Folgen einer Handlung, bevor verspätetes Feedback verfügbar ist. Dadurch kann das motorische System schnelle Bewegungen kontrollieren, ohne auf langsame Rückmeldeschleifen warten zu müssen.

## Efferenzkopie

Jäncke beschreibt im erweiterten Reafferenzprinzip, dass von jeder Efferenz eine [[efferenzkopie|Efferenzkopie]] angefertigt wird. Mit gespeicherten Informationen kann das System daraus erwartete Afferenzen vorhersagen.

## Regelungstechnische Funktion

Karnath definiert das Vorwärtsmodell als Lösung für das Verzögerungsproblem in Regelkreisen: Wenn der aktuelle Istwert erst verspätet verfügbar ist, ersetzt das Modell ihn durch eine zeitnahe Schätzung des erwarteten Istwertes.

## Merkpunkte

- Sagt sensorische Konsequenzen eigener Handlungen voraus.
- Nutzt Efferenzkopie und gespeicherte Zustandsinformationen.
- Brücke zwischen Open-loop- und Closed-loop-Kontrolle.
