# Orientierungskolumne

Eine Orientierungskolumne ist eine Säule von Neuronen in der Sehrinde, die über mehrere kortikale Schichten hinweg auf ähnliche Stimulusorientierungen maximal reagieren.

## Merkpunkte

- Orientierungskolumnen organisieren lokale V1-Verarbeitung.
- Sie bündeln Neurone mit ähnlicher Orientierungspräferenz.
