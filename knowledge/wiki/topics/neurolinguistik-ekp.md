# Neurolinguistik: EKP-Komponenten der Sprachverarbeitung

## Grundlagen: Ereigniskorrelierte Hirnpotenziale (EKP)

**EKP** (engl. ERP) = gemittelte elektrische Hirnaktivität, gebunden an ein definiertes sprachliches Ereignis (Wort, Satz). Variieren in:
- **Latenz** (wann?)
- **Polarität** (negativ / positiv)
- **Amplitude** (wie stark?)
- **Topografie** (wo an der Kopfoberfläche?)

Vorteil gegenüber fMRT: Zeitauflösung im **Millisekundenbereich** — ermöglicht Sequenzierung von Sprachverarbeitungsschritten.

## Die drei Hauptkomponenten

### 1. N400 — Semantische Verarbeitung

| Merkmal | Beschreibung |
|---------|-------------|
| **Polarität / Latenz** | Negativierung, Peak ~400 ms |
| **Topografie** | Zentral und posterior |
| **Auslöser** | Semantisch unpassende oder unerwartete Wörter im Kontext |
| **Amplitude** | Je geringer die semantische Passung, desto größer die N400 |
| **Erstbeschreibung** | Kutas & Hillyard (1980) |

Die N400 ist **sensitiv für**:
- Semantisch-pragmatische Verletzungen (Plausibilität)
- Semantisch-lexikalische Verletzungen (Selektionsbeschränkungen)
- Einfache Wortkontexte (semantisches Priming: kleinere N400 nach relatierten Wörtern)
- Wortklasse: **Inhaltswörter** → zentro-posteriore N400 (200–500 ms); **Funktionswörter** → frontale Negativierung (400–700 ms) + frühere N280 links-anterior

### 2. ELAN / LAN — Syntaktische Erstanalyse

**ELAN** (Early Left Anterior Negativity):

| Merkmal | Beschreibung |
|---------|-------------|
| **Polarität / Latenz** | Negativierung, **100–200 ms** |
| **Topografie** | Links-anterior |
| **Auslöser** | Phrasenstrukturverletzungen (falsche Wortkategorie, z. B. „der Freund wurde im besucht") |
| **Bedeutung** | Reflektiert initiale syntaktische Kategorisierung |

**LAN** (Left Anterior Negativity):

| Merkmal | Beschreibung |
|---------|-------------|
| **Latenz** | 300–500 ms |
| **Topografie** | Links-anterior / links-fronto-temporal |
| **Auslöser** | Morphosyntaktische Verletzungen (Subjekt-Verb-Kongruenz, z. B. „er singen") |
| **Abgrenzung** | LAN bei Kongruenzfehlern; andere LAN-artige Komponenten bei Arbeitsgedächtnisbelastung (globales langsames Potenzial über gesamten Satz — kein nachfolgender P600) |

### 3. P600 — Syntaktische Integration und Reanalyse

| Merkmal | Beschreibung |
|---------|-------------|
| **Polarität / Latenz** | Positivierung, ~600 ms (und später) |
| **Topografie** | Zentro-parietal |
| **Auslöser** | Syntaktisch inkorrekte Sätze; komplexe oder nichtpräferierte Strukturen; „Holzweg"-Sätze |
| **Interpretationen** | Syntaktische Revisionsprozesse (Friederici); Integration von Syntax und Semantik (Kuperberg) |

## Neurokognitives Modell der Sprachverarbeitung (Friederici 1995, 2002)

Drei zeitlich aufeinanderfolgende Phasen:

```
Phase 1 (150–250 ms) → ELAN
  Initiale Phrasenstrukturerstellung
  auf Basis von Wortkategorieinformation (Nomen, Verb, Konjunktion)

Phase 2 (300–500 ms) → N400 / LAN
  Lexikalisch-semantische Verarbeitung
  + Zuweisung thematischer Rollen

Phase 3 (>500 ms) → P600
  Syntaktische Integration, Abgleich, Reanalyse
```

## Topografie der Sprachverarbeitung (fMRT)

| Funktion | Hauptareale |
|---------|------------|
| Phonologische Diskrimination | Bilateral Gyrus temporalis superior (BA 42, 22); bei Diskriminationsaufgaben links-dominant |
| Semantische Verarbeitung | Linker inferiorer frontaler Kortex (BA 45, 47); linker mittlerer temporaler Gyrus |
| Syntaktische Verarbeitung | Linker Gyrus frontalis inferior (BA 44 = Broca-Areal) + frontales Operculum; vorderer linker superiorer Temporalgyrus |
| Prosodie | Rechter superiorer Temporalgyrus; rechtes frontales Operculum |

**Kernaussage:** Links-inferior-frontale (BA 44/45/47) + links-superior-temporale Areale bilden das zentrale Sprachverarbeitungsnetzwerk; prosodische Verarbeitung rechtshemisphärisch dominant.

## Besonderheiten

- **BA 44 (Broca)** ist involviert bei syntaktischer Verarbeitung, phonologischen Kontrollprozessen, Verarbeitung musikalischer Syntax und komplexen regelbasierten motorischen Sequenzen
- Syntaktische Arbeitsgedächtnisanforderungen (lange Relativsätze) erhöhen BA 44-Aktivierung in enger Zusammenarbeit mit dem inferioren frontalen Sulcus
- **Fasciculus arcuatus** verbindet Broca- und Wernicke-Areal (dorsaler Sprachpfad)

## Prüfungsrelevanz
- N400: Semantik, ~400 ms, zentro-posterior, Kutas & Hillyard
- ELAN: Phrasenstruktur, 100–200 ms, links-anterior
- LAN: Morphosyntax, 300–500 ms, links-anterior; abzugrenzen von Arbeitsgedächtnis-LAN
- P600: Reanalyse, >500 ms, zentro-parietal
- Drei-Phasen-Modell Friederici (ELAN → N400/LAN → P600)
- Topografie: BA 44/45 syntaktisch-phonologisch; BA 45/47 semantisch; Prosodie rechts-temporal
## Merkpunkte

- Neurolinguistik/EKP verbindet Sprachsysteme mit elektrophysiologischen Messungen.
- ERP-Komponenten wie N400, ELAN und P600 sind die wichtigste Brücke zwischen Linguistik und Neuropsychologie.
