# Ensemble-Code

Ein Ensemble-Code ist die Repräsentation sensorischer, motorischer oder kognitiver Information durch Aktivität, die über eine große Zahl von Neuronen verteilt ist.

## Merkpunkte

- Der Begriff betont verteilte Aktivitätsmuster statt einzelner Neurone.
- Bear nennt Farbcodierung durch relative Aktivität verschiedener Zapfentypen als Beispiel.
