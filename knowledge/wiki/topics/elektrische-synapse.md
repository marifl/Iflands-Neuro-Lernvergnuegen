# Elektrische Synapse

Eine elektrische Synapse ist eine Synapse, bei der elektrischer Strom direkt über Gap Junctions von einer Zelle zur nächsten fließt.

## Merkpunkte

- Elektrische Synapsen übertragen direkt elektrisch.
- Sie unterscheiden sich von chemischen Synapsen durch den Stromfluss über Gap Junctions.
