# Brechung

Brechung ist die Richtungsänderung von Lichtstrahlen beim Übergang zwischen transparenten Medien. Für das visuelle System ist sie grundlegend, weil Hornhaut und Linse Licht so bündeln, dass ein scharfes Bild auf der Retina entstehen kann.

## Merkpunkte

- Brechung ist ein optischer Grundlagenprozess.
- Störungen der Brechkraft verändern die Bildschärfe, bevor kortikale Verarbeitung beginnt.
