# Parallele Reizverarbeitung

Parallele Reizverarbeitung beschreibt die Vorstellung, dass unterschiedliche Reizmerkmale im Gehirn gleichzeitig in getrennten Bahnen verarbeitet werden.

## Merkpunkte

- Im visuellen System betrifft das unter anderem Form, Farbe, Bewegung und Raum.
- Das Konzept erklärt Spezialisierung, erzeugt aber auch das Bindungsproblem.
