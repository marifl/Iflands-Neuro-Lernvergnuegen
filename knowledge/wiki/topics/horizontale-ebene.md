# Horizontale Ebene

Die horizontale Ebene ist eine anatomische Schnittebene, die den Körper oder das Gehirn in obere und untere Abschnitte gliedert.

## Merkpunkte

- Synonym wird häufig Transversalebene verwendet.
- Sie ist eine Raumorientierung, keine eigenständige Struktur.
