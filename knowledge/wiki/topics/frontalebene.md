# Frontalebene

Die Frontalebene ist eine anatomische Schnittebene parallel zur Stirn. Sie unterteilt das Nervensystem in einen anterioren und einen posterioren Bereich.

## Merkpunkte

- Die Frontalebene ist eine Standardebene der Neuroanatomie.
- Sie wird auch Koronalebene genannt.
