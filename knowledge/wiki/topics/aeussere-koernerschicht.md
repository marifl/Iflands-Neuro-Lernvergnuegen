# Äußere Körnerschicht

Die äußere Körnerschicht ist eine Schicht der Netzhaut, die die Zellkörper der Photorezeptoren enthält.

## Merkpunkte

- Sie ist eine retinale Zellkörperschicht.
- Ihr Inhalt grenzt sie von plexiformen Synapsenschichten ab.
