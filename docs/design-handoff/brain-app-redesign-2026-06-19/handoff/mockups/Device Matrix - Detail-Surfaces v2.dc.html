Sechs Detail-Surfaces auf allen Geräten

Jede Surface ist ein eigenes responsives Component, das seine Containergröße misst und das Layout per Tier wählt. Regel: w < 820 (Phone + iPad Mini Portrait) → STACK · w ≥ 820 (iPad Pro Portrait + alle Landscape + Desktop) → COLUMNS · h < 480 (Phone Landscape) → kompakt. iPad Mini Portrait ≠ iPad Pro Portrait.
